# Hirelytics - HR Data Analysis Project

Hirelytics is a comprehensive HR analytics project focused on exploring employee data using Python for Exploratory Data Analysis (EDA) and visualization techniques. The project uncovers key workforce insights such as salary distribution, department trends, correlations, and outliers.

---

## 📌 Project Overview

This project performs detailed analysis on HR data to understand employee demographics, salary patterns, departmental performance, and workforce structure using statistical and visual methods.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 Analysis Performed

### 1. Univariate Analysis
- Distribution of numerical variables
- Histograms and density plots
- Understanding data spread and skewness

### 2. Categorical Analysis
- Gender distribution
- Department-wise employee count
- Bar chart visualizations

### 3. Bivariate Analysis
- Salary vs Gender comparison
- Salary vs Department
- Box plots for trend analysis

### 4. Correlation Analysis
- Heatmap to identify relationships between variables
- Understanding feature dependencies

### 5. Feature Analysis
- Salary distribution insights
- Age and tenure impact

### 6. Outlier Detection
- Boxplots to detect anomalies
- Identification of extreme values

### 7. Pairplot Analysis
- Multi-variable relationship visualization
- Pattern and trend discovery

---

## 📊 Key Insights

- Salary distribution follows a near-normal pattern  
- Certain departments have higher employee concentration  
- Salary variations exist across gender and departments  
- Strong correlations found between key numerical variables  
- Outliers detected in salary and age distributions
